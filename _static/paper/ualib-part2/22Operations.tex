\textbf{Notation}. In the line above, we renamed for notational
convenience the \texttt{generator} constructor of the \texttt{Term}
type, so from now on we use \ab{ℊ} in place of \texttt{generator}.

When we interpret a term in an algebra we call the resulting function a
\defn{term operation}. Given a term \ab{𝑝} and an algebra
\ab{𝑨}, we denote by \ab{𝑝\ ̇\ 𝑨} the \defn{interpretation} of
\ab{𝑝} in \ab{𝑨}. This is defined inductively as follows.

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\item
  If \ab{𝑝} is a variable symbol \ab{x\ :\ X} and if
  \ab{𝒂\ :\ X\ →\ ∣\ 𝑨\ ∣} is a tuple of elements of
  \texttt{∣\ 𝑨\ ∣}, then \ab{(𝑝\ ̇\ 𝑨)\ 𝒂\ :=\ 𝒂\ x}.
\item
  If \ab{𝑝\ =\ 𝑓\ 𝑡}, where \ab{𝑓\ :\ ∣\ 𝑆\ ∣} is an operation
  symbol, if \ab{𝑡\ :\ ∥\ 𝑆\ ∥\ 𝑓\ →\ 𝑻\ X} is a tuple of terms, and
  if \ab{𝒂\ :\ X\ →\ ∣\ 𝑨\ ∣} is a tuple from \ab{𝑨}, then we
  define
  \ab{(𝑝\ ̇\ 𝑨)\ 𝒂\ =\ (𝑓\ 𝑡\ ̇\ 𝑨)\ 𝒂\ :=\ (𝑓\ ̂\ 𝑨)\ λ\ i\ →\ (𝑡\ i\ ̇\ 𝑨)\ 𝒂}.
\end{enumerate}

Thus the interpretation of a term is defined by induction on the
structure of the term, and the definition is formally implemented in the
{[}UALib{]}{[}{]} as follows.

What if the algebra 𝑨 in question happens to be \texttt{𝑻\ X} itself?
Assume the map \ab{h\ :\ X\ →\ ∣\ 𝑻\ X\ ∣} is the identity. We
expect that \texttt{∀\ 𝑠\ →\ (p\ ̇\ 𝑻\ X)\ 𝑠\ \ ≡\ \ p\ 𝑠}. But what is
\texttt{(𝑝\ ̇\ 𝑻\ X)\ 𝑠} exactly?

By definition, it depends on the form of 𝑝 as follows:

\begin{itemize}
\item
  if \ab{𝑝\ =\ ℊ\ x}, then
  \texttt{(𝑝\ ̇\ 𝑻\ X)\ 𝑠\ :=\ (ℊ\ x\ ̇\ 𝑻\ X)\ 𝑠\ ≡\ 𝑠\ x}
\item
  if \texttt{𝑝\ =\ node\ 𝑓\ 𝑡}, then
  \texttt{(𝑝\ ̇\ 𝑻\ X)\ 𝑠\ :=\ (node\ 𝑓\ 𝑡\ ̇\ 𝑻\ X)\ 𝑠\ =\ (𝑓\ ̂\ 𝑻\ X)\ λ\ i\ →\ (𝑡\ i\ ̇\ 𝑻\ X)\ 𝑠}
\end{itemize}

Now, assume \ab{ϕ\ :\ hom\ 𝑻\ 𝑨}. Then by \texttt{comm-hom-term}, we
have \texttt{∣\ ϕ\ ∣\ (p\ ̇\ 𝑻\ X)\ 𝑠\ =\ (p\ ̇\ 𝑨)\ ∣\ ϕ\ ∣\ ∘\ 𝑠}.

\begin{itemize}
\tightlist
\item
  if \ab{p\ =\ ℊ\ x}, then
\end{itemize}

∣ ϕ ∣ p ≡ ∣ ϕ ∣ (ℊ x) ≡ ∣ ϕ ∣ (λ h → h x) (where h : X → ∣ 𝑻(X) ∣ ) ≡ λ
h → (∣ ϕ ∣ ∘ h) x

\begin{itemize}
\tightlist
\item
  if \ab{p\ =\ node\ 𝑓\ 𝑡}, then
\end{itemize}

∣ ϕ ∣ p ≡ ∣ ϕ ∣ (p ̇ 𝑻 X) 𝑠 = (node 𝑓 𝑡 ̇ 𝑻 X) 𝑠 = (𝑓 ̂ 𝑻 X) λ i → (𝑡 i ̇ 𝑻
X) 𝑠

We claim that for all \ab{p\ :\ Term\ X} there exists
\ab{q\ :\ Term\ X} and \ab{h\ :\ X\ →\ ∣\ 𝑻\ X\ ∣} such that
\ab{p\ ≡\ (q\ ̇\ 𝑻\ X)\ h}. We prove this fact as follows.
