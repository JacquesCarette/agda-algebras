These types should be self-explanatory, but just to be sure, let's
describe the Sigma type appearing in the second definition. Given an
\ab{𝑆}-algebra \ab{𝑨\ :\ Algebra\ 𝓤\ 𝑆}, the type
\texttt{HomImagesOf\ 𝑨} denotes the class of algebras
\ab{𝑩\ :\ Algebra\ 𝓦\ 𝑆} with a map
\ab{φ\ :\ ∣\ 𝑨\ ∣\ →\ ∣\ 𝑩\ ∣} such that \ab{φ} is a surjective
homomorphism.

Since we take the class of homomorphic images of an algebra to be closed
under isomorphism, we consider \ab{𝑩} to be a homomorphic image of
\ab{𝑨} if there exists an algebra \ab{𝑪} which is a homomorphic
image of \ab{𝑨} and isomorphic to \ab{𝑩}. The following type
expresses this.
