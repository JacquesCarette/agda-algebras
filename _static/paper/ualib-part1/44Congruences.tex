Recall that a \defn{congruence relation} of an algebra \ab{𝑨} is defined to be an equivalence relation that is compatible with the basic operations of \ab 𝑨. This concept can be represented in a number of alternative but equivalent ways.  % Informally, a relation is a congruence of \ab 𝑨 provided it is both an equivalence relation on the domain of \ab 𝑨 and a subalgebra of the square of \ab 𝑨.  
Formally, we define a record type (\ar{IsCongruence}) to represent the \defn{property of being a congruence}, and we define a Sigma type (\af{Con}) to represent the \defn{type of congruences} of a given algebra.
