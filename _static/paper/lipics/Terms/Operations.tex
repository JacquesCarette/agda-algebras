When we interpret a term in an algebra we call the resulting function a \textbf{term operation}. Given a term \ab 𝑝\ \as : \af{Term} and an algebra \ab 𝑨, we denote by \ab 𝑝 \af ̇ \ab 𝑨 the \textbf{interpretation} of \ab 𝑝 in \ab 𝑨.
This is defined inductively as follows:
\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  if \ab 𝑝 is \ab x \as : \ab X (a variable) and if \ab 𝒂 \as : \ab X \as → \aiab{∣}{𝑨} is a tuple of elements of
  \aiab{∣}{𝑨}, then define (\ab 𝑝 \af ̇ \ab 𝑨) \ab 𝒂 = \ab 𝒂 \ab x;
\item
  if \ab 𝑝 \as = \ab 𝑓 \ab 𝒔, where \ab 𝑓 \as : \aiab{∣}{𝑆} is an operation symbol and  \ab 𝒔 \as : \aiab{∥}{𝑆} \ab f \as → \af 𝑻 \ab X is an (\aiab{∥}{𝑆} \ab f)-tuple of terms and \ab 𝒂 \as : \ab X \as → \aiab{∣}{𝑨} is a tuple from \ab 𝑨, then define (\ab 𝑝 \af ̇ \ab 𝑨) \ab 𝒂 = ((\ab 𝑓 \ab 𝒔) \af ̇ \ab 𝑨) \ab 𝒂 = (\ab 𝑓 \af ̂ \ab 𝑨) \as λ \ab i \as → ((\ab 𝒔 \ab i) \af ̇ \ab 𝑨) \ab 𝒂
\end{enumerate}

Here we want (\ab 𝒕 \as : \ab X \as → \aiab{∣}{𝑻(X)}) \as → ((\ab p \af ̇ \af 𝑻(\ab X)) \ab 𝒕) \ad ≡ \ab p \ab 𝒕 …, but what is (\ab 𝑝 \af ̇ \af 𝑻(\ab X)) \ab 𝒕?  By definition, it depends on the form of \ab 𝑝 as follows:
\begin{itemize}
\item
  if \ab 𝑝 \as = (\aic{generator} \ab x), then (\ab 𝑝 \af ̇ \af 𝑻(\ab X)) \ab 𝒕 \as = ((\aic{generator} \ab x) \af ̇ \af 𝑻(\ab X)) \ab 𝒕 \as = \ab 𝒕 \ab x
\item
  if \ab 𝑝 \as = (\aic{node} \ab f \ab{args}), then (\ab 𝑝 \af ̇ \af 𝑻(\ab X)) \ab 𝒕 \as = ((\aic{node} \ab f \ab{args}) \af ̇ \af 𝑻(\ab X)) \ab 𝒕 \as = (\ab f \af ̂ \af 𝑻(\ab X)) \as λ \ab i \as → (\ab{args} \ab i \af ̇ \af 𝑻(\ab X)) \ab 𝒕.
\end{itemize}
Let \ab h \as : \af{hom} \af 𝑻 \ab 𝑨. Then by \af{comm-hom-term}, \aiab{∣}{h} (\ab p \af ̇ \af 𝑻(\ab X)) \ab 𝒕 \as = (\ab p \af ̇ \ab 𝑨) \aiab{∣}{h} \af ∘ \ab 𝒕.
\begin{itemize}
\item
  if \ab p \as = (\aic{generator} \ab x), then
\end{itemize}
\aiab{∣}{h} \ab p \ad ≡ \aiab{∣}{h} (\aic{generator} \ab x) \ad ≡ \as λ \ab 𝒕 \as → \ab 𝒕 \ab x) (where \ab 𝒕 \as : \ab X \as → \aiab{∣}{𝑻(X)}) \ad ≡ (\as λ \ab 𝒕 \as → (\aiab{∣}{h} \af ∘ \ab 𝒕) \ab x)

\aiab{∣}{h} \ab p \ad ≡ \aiab{∣}{h} (\as λ \ab 𝒕 \as → \ab 𝒕 \ab x) (where \ab 𝒕 \as : \ab X \as → \aiab{∣}{𝑻(X)}) \ad ≡ (\as λ \ab 𝒕 \as → (\aiab{∣}{h} \af ∘ \ab 𝒕) \ab x)

\begin{itemize}
\item if \ab p \as = (\aic{node} \ab f \ab{args}), then
\end{itemize}

\aiab{∣}{h} \ab p \ad ≡ \aiab{∣}{h} (\ab p \af ̇ \af 𝑻(\ab X)) \ab 𝒕 \as = ((\aic{node} \ab f \ab{args}) \af ̇ \af 𝑻(\ab X)) \ab 𝒕 \as = (\ab f \af ̂ \af 𝑻(\ab X)) \as λ \ab i \as → (\ab{args} \ab i \af ̇ \af 𝑻(\ab X)) \ab 𝒕.
